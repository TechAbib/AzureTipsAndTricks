          # Prebuilt Azure VMs ready for Containers

          While I was at Microsoft Connect last week, I met someone who recently read my Docker Container mini-series that covered Getting Started with Docker and Azure, Running an app inside a Container Image with Docker, Creating a Container Image with Docker and Pushing a Container Image to a Docker Repo. The question that came up was how they wish we had prebuilt Azure VMs that had container images already on them. This would save them additional time and something that they would expect. You can imagine their surprise when I told them that we had them already. In this post, we'll take a quick look at how to find them.

          Open Virtual Machines and click on Add, then search for containers. You may optionally want to set a filter to restrict to Windows and Linux based.

          From this screen, you can see a variety of Windows and Linux VMs. If I wanted a Windows VM, then I'd probably pick this one:

          Note the following text: It includes Windows Server container images installed and ready to use with Docker.
